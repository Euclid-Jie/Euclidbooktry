# 正则语句

## 元字符

```python
[A-Z]    # 匹配从A到Z的所有大写字母
[a-z]    # 匹配从a到z的所有小写字母
[A-z]    # 匹配大小写字母以及[,^等ASCII字符表中的任意字符
[0-9]    # 任何一个数字字符
[A-Z0-9] # 匹配大写字母及数字
\d       # 任何一个数字字符，等价于[0-9]
\D       # 任何一个非数字字符，等价于[^0-9]
\W       # 任何一个字母数字字符（大小写均可）及下划线，等价于[a-zA-Z0-9_]
\w       # 等价于[^a-zA-Z0-9_]

```

## 重复匹配

```python
+      # 表示匹配一个或多个（至少一个）
?      # 表示匹配零个或者一个
*      # 表示有或者没有
{3}    # 表示重复三次
{3-5}  # 表示重复次数在3-5之间
{3，}  # 表示重复三次及以上
```

### 懒惰匹配

```python
+?     # +的懒惰模式
*?     # *的懒惰模式
{n,}   # {n,}的懒惰模式
```
