- 初始化一个仓库

  ```bash
  git init
  ```

- 查看远程信息

  ```bash
  git remote -v # 查看远程仓库地址
  git remote # 查看远程仓库分支
  git remote show origin # 查看某个分支的具体信息
  ```

- 储存 

  不常用，主要是为了处理冲突

  ```bash
  git stash save "本地缓存内容标识" # 创建储存
  git stash list # 查看有哪些储存
  git stash clear # 清除所有储存
  ```
